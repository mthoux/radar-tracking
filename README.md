# Radar Tracking

A radar tracking project using radar data processing techniques.

## Hardware 

2 FMCW TI 77GHz Radars.

## Getting Started

Clone the repository:

git clone https://github.com/mthoux/radar-tracking.git
cd radar-tracking

Install dependencies:

pip install -r requirements.txt

Run the project:

## Project Structure
```
radar-tracking/
├── data/           # Radar samples and datasets
├── src/            # Core source code
├── models/         # Tracking or detection models
├── scripts/        # Utility scripts
└── main.py         # Entry point
```
